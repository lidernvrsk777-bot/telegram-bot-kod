# Telegram Bot — КОД УПРАВЛЕНИЯ

Telegram бот для проекта «КОД УПРАВЛЕНИЯ» — программы обучения управлению кафе за 30 дней.

## Возможности

- ✅ Ответы на вопросы о программе
- ✅ Информация о результатах
- ✅ Сведения об авторе (Aslan Uzhakhov)
- ✅ Перевод на оператора при необходимости
- ✅ 24/7 работа на Render

## Развёртывание

Бот развёрнут на Render и работает 24/7.

### Переменные окружения

- `TELEGRAM_BOT_TOKEN` — токен бота от @BotFather
- `TELEGRAM_CHAT_ID` — ID для отправки уведомлений

## Локальный запуск

```bash
pip install -r requirements.txt
python bot.py
```
